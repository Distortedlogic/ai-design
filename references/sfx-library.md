## Usage conventions

### Volume guidelines (from the audio dual-track scheme in apple-gallery-showcase.md)
- **SFX main track**: `1.0` (no attenuation)
- **BGM background track**: `0.4 ~ 0.5` (SFX clearly cuts through)
- **Stacking multiple SFX**: use `amix=inputs=N:duration=longest:normalize=0` to preserve dynamic range

### ffmpeg merging template
```bash
# single SFX aligned to a time point:
ffmpeg -i video.mp4 -itsoffset 2.5 -i sfx/ui/click.mp3 \
  -filter_complex "[0:a][1:a]amix=inputs=2:duration=longest:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4

# multiple SFX + BGM:
ffmpeg -i video.mp4 \
  -itsoffset 1.0 -i sfx/transition/whoosh-fast.mp3 \
  -itsoffset 1.6 -i sfx/impact/logo-reveal-v2.mp3 \
  -i bgm.mp3 \
  -filter_complex "[3:a]volume=0.4[bgm];[0:a][1:a][2:a][bgm]amix=inputs=4:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4
```

### Selection decision tree
1. **Tactile action** (typing / clicking / sliding) → `keyboard/` or `ui/`
2. **Element entry / exit** → `transition/`
3. **Container-layer operations** (cards / modals) → `container/`
4. **State feedback** (success / failure / notification) → `feedback/`
5. **Progress / time elapsing** → `progress/`
6. **Brand landing / important moment** → `impact/`
7. **AI magic / transformation** → `magic/`
8. **Command line / code demo** → `terminal/`

### Avoid stacked-sound pile-up
- At any single time point, `max 2 concurrent SFX`
- When BGM drops below 0.3, you can play 3
- For brand impact moments, clear all other SFX (leave 0.2s of silence before the landing)

## Prompt-writing principles (for reuse)

Reference style: `apple keynote, tight, minimal, no reverb unless ambient, crisp, elegant`

**The three elements of a good prompt**:
1. **Physical sound description**: what object, what action ("mechanical keyboard single key press")
2. **Texture / style qualifiers**: apple-style / ios-style / cinematic / retro
3. **Counter-examples to exclude**: no reverb / clean studio / minimal

❌ "click sound"
✅ "crisp ui button click, clean modern interface sound, apple-style, high pitched"

❌ "magic"
✅ "bright twinkling stars sound, high pitched glittery chime, fairy dust"
